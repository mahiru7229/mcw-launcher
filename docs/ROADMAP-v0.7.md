# MCW Launcher 0.7 Roadmap

- CurseForge Gateway integration
- CurseForge mod search
- CurseForge file/version selection
- Automatic download when permitted
- Manual download fallback
- Local JSON cache
- 10 MB cache size limit

- Request cooldown and deduplication

- GitHub Releases storage

- Multiple download mirrors
- Cloudflare R2
- MCW server provider
- Automatic fallback
